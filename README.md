# MedMind

MedMind is an AI-powered health assistant backend.

It combines BERT-based intent classification, RAG-based medical document retrieval, FAISS vector search, OpenAI function calling, session memory, appointment handling, and PostgreSQL integration.

## Status

MedMind is currently under development.

## Privacy Note

Real medical PDFs, generated FAISS indexes, trained BERT model files, datasets, checkpoints, and environment files are excluded from this repository.
